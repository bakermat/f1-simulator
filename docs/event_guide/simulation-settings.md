# Simulation Settings

1. Select **SIMULATION SETTINGS**
2. Configure the simulation settings:

    <pre>
    Recovery Mode: **Auto-Recovery**
    Surface Type: **Realistic**
    Low Fuel Mode: **Easy**
    Race Starts: **Assisted**
    Car Damage: **Off**
    Collisions: **On**
    </pre>

3. Return to the **Grand Prixâ„¢ Settings** screen
